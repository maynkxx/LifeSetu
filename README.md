## Technical Architecture
LifeSetu is built as a **Multi-tenant SaaS Platform** utilizing the **MERN Stack** to ensure high availability and rapid response times.

* **Frontend:** React.js for a responsive, real-time dashboard.
* **Backend:** Node.js & Express.js implementing a Service-Repository pattern.
* **Database:** MongoDB for flexible, geo-spatial resource storage.
* **Model:** SaaS (Software as a Service) to support multiple independent relief organizations.
